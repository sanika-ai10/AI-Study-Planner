# AI Study Planner

## Overview

AI Study Planner is a Python-based mini project designed to help students organize and manage their study schedules efficiently.

The system allows users to add study tasks, prioritize subjects based on difficulty and study hours, generate a study timetable, monitor progress, calculate attendance percentage, and receive study recommendations.

---

## Features

### Study Task Management
- Add study tasks
- Enter subject name
- Enter topic name
- Assign study hours
- Select difficulty level

### Task Tracking
- View all tasks
- Mark tasks as completed
- Display pending and completed tasks

### Dashboard
- Total tasks
- Completed tasks
- Planned study hours
- Progress percentage

### AI Recommendation
- Analyzes difficulty levels
- Suggests study strategies
- Encourages consistent learning

### Timetable Generator
- Sorts tasks based on priority
- Generates an optimized study schedule

### Attendance Calculator
- Calculates attendance percentage
- Predicts classes required to reach 75% attendance

### Motivation Section
- Displays motivational quotes
